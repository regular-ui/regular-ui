### 示例
#### 基本形式

<div class="m-example"></div>

```html
<ul class="m-sidebar">
    <li><a>Item</a></li>
    <li class="z-crt"><a>Current</a></li>
    <li><a>Item</a></li>
    <li><span>Text</span></li>
</ul>
```

#### 分组

<div class="m-example"></div>

```html
<ul class="m-sidebar">
    <li><a>Item</a></li>
    <li><a>Item</a></li>
    <li><a>Item</a></li>
    <li class="sidebar_group"><span>Group 1</span></li>
    <li><a>Item</a></li>
    <li class="z-crt"><a>Current</a></li>
    <li><span>Text</span></li>
    <li class="sidebar_group"><span>Group 2</span></li>
    <li><a>Item</a></li>
    <li><a>Item</a></li>
   <li><span>Text</span></li>
</ul>
```

#### 分隔符

<div class="m-example"></div>

```html
<ul class="m-sidebar">
    <li><a>Item</a></li>
    <li class="z-crt"><a>Current</a></li>
    <li><a>Item</a></li>
    <li class="sidebar_divider"></li>
    <li><a>Item</a></li>
    <li><a>Item</a></li>
   <li><span>Text</span></li>
</ul>
```
