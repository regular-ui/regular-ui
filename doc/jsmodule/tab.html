选项卡 Tab

示例

基本形式

<tab>
    <tabPane name="tab1">content1</tabPane>
    <tabPane name="tab2">content2</tabPane>
    <tabPane name="tab3" disabled={true}>content3</tabPane>
    <tabPane name="tab4">content4</tabPane>
</tab>

居中

<tab class="m-tab-center">
    <tabPane name="tab1">content1</tabPane>
    <tabPane name="tab2">content2</tabPane>
    <tabPane name="tab3" disabled={true}>content3</tabPane>
    <tabPane name="tab4">content4</tabPane>
</tab>

垂直居左

<tab class="m-tab-left">
    <tabPane name="tab1">content1</tabPane>
    <tabPane name="tab2">content2</tabPane>
    <tabPane name="tab3" disabled={true}>content3</tabPane>
    <tabPane name="tab4">content4</tabPane>
</tab>

垂直居右

<tab class="m-tab-right">
    <tabPane name="tab1">content1</tabPane>
    <tabPane name="tab2">content2</tabPane>
    <tabPane name="tab3" disabled={true}>content3</tabPane>
    <tabPane name="tab4">content4</tabPane>
</tab>

禁用

<tab disabled={true}>
    <tabPane name="tab1">content1</tabPane>
    <tabPane name="tab2">content2</tabPane>
    <tabPane name="tab3">content3</tabPane>
    <tabPane name="tab4">content4</tabPane>
</tab>

API

Class

Tab

继承自Component。

Options

new Tab({options})
参数类型默认值描述
dataobject绑定属性
data.readonlybooleanfalse是否只读
data.disabledbooleanfalse是否禁用
data.visiblebooleantrue是否显示
data.classstring''补充class

Methods

tab.select(item)

选择某一项

参数类型默认值描述
itemobject选择项

Events

on-select={$event}

选择某一项时触发

属性类型描述
selectedobject当前选择项
