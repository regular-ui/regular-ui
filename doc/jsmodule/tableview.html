示例

基本形式

<tableView source={source} fields={@(fields)} itemTemplate={@(itemTemplate)} />

var component = new RGUI.Component({
    template: template,
    data: {
        source: [
        {id: 3, name: '精通Javascript开发', org: '前端Funs', hits: 42371},
        {id: 2, name: 'Android深入浅出', org: 'Android学院', hits: 30645},
        {id: 1, name: 'cocos2dx游戏开发教程', org: '鱼C课程', hits: 25112},
        {id: 4, name: 'MySQL数据库', org: 'LAMP兄弟连', hits: 18089},
        {id: 5, name: 'Arduino初级教程', org: '硬件社', hits: 16361}
    ];

    this.data.fields = [
        {key: 'name', name: '课程'},
        {key: 'org', name: '机构'},
        {key: 'hits', name: '点击率'}
    ];

    this.data.itemTemplate = _.multiline(function(){/*
<div class="u-courceitem">
    <div><img src="{item.img}" alt="{item.name}" title="{item.name}"></div>
    <div><a href="#">{item.name}</a></div>
    <div>
        <span class="org">{item.org}</span>
        <span class="hits f-fr"><i class="u-icon u-icon-user"></i> {item.hits}</span>
    </div>
</div>
    */})
    }
});

排序

只要给可排序的字段添加sortable: true即可。

如果有service属性，会向服务器发送请求并带上排序相关的参数，进行后端排序；如果没有service属性，则在本地对source进行排序。

API

Class

TableView

继承自SourceComponent。

Options

new TableView()

参数类型默认值数据流向描述
dataobject绑定属性
data.sourceobject[][]outer <=> inner数据源
data.source[].namestringouter => inner每项的内容
data.fieldobject[][]outer => inner字段集
data.field[].keystringouter => inner每个字段的key
data.field[].namestringouter => inner每个字段在表头显示的文字，如果没有则显示key
data.stripedbooleanfalseouter => inner是否显示条纹
data.hoverbooleanfalseouter => inner是否每行在hover时显示样式
data.visiblebooleantrueouter => inner是否显示
data.classstring''outer => inner补充class
serviceobjectouter @=> inner数据服务

Methods

tableView.sort(field)

按照某个字段排序

参数类型默认值描述
fieldobject排序字段
返回值类型描述
无返回值

Inherited Methods

@deprecated tableView.$updateSource()

从service中更新数据源

参数类型默认值描述
无参数
返回值类型描述
thisSourceComponent

Events

tableView.$on('sort')

按照某个字段排序时触发

属性类型描述
senderobject事件发送对象
fieldobject排序字段
