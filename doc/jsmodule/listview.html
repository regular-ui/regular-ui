列表视图 ListView

示例

基本形式

<listView source={source} />

...
    this.data.source = [
        {id: 1, name: '选项1'},
        {id: 2, name: '选项2'},
        {id: 3, name: '选项3'}
    ];
...

禁用组件

<listView source={source} disabled={true} />

...
    this.data.source = [
        {id: 1, name: '选项1'},
        {id: 2, name: '选项2'},
        {id: 3, name: '选项3'}
    ];
...

远程数据

<listView service={@(service)} />

...
    this.data.service = {
        getList: function(params, success) {
            ajax.request({
                url: 'listbox.json',
                method: 'get',
                type: 'json',
                success: success
            });
        }
    };
...

API

Class

ListView

继承自。

Options

new ListView()
参数类型默认值描述
dataobject绑定属性
data.sourceobject[][]数据源
data.source[].idnumber每项的id
data.source[].namestring每项的内容
data.selectedobjectnull当前选择项
data.itemTemplatestringnull单项模板
data.readonlybooleanfalse是否只读
data.disabledbooleanfalse是否禁用
data.visiblebooleantrue是否显示
data.classstring''补充class
serviceobject数据服务

Methods

listView.select(item)

选择某一项

参数类型默认值描述
itemobject选择项

Events

listView.$on('select')

选择某一项时触发

属性类型描述
selectedobject当前选择项
