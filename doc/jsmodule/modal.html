模态对话框 Modal

含有遮罩层的对话框，用于模拟浏览器的alert、confirm和prompt。

模态对话框通过遮罩层来阻止用户的其他行为。

示例

基本形式

<button class="u-btn u-btn-primary" on-click={this.showModal()}>Modal</button>

var component = new RGUI.Component({
    template: template,
    showModal: function() {
        var modal = new RGUI.Modal({
            data: {
                title: 'Modal标题',
                content: 'Modal内容'
            }
        });
    }
});

Alert

<button class="u-btn u-btn-error" on-click={this.showModal()}>Alert</button>

var component = new RGUI.Component({
    template: template,
    showModal: function() {
        RGUI.Modal.alert('Alert内容');
    }
});

Confirm

<button class="u-btn u-btn-success" on-click={this.showModal()}>Confirm</button>

var component = new RGUI.Component({
    template: template,
    showModal: function() {
        RGUI.Modal.confirm('Confirm内容');
    }
});

API

Class

Modal

继承自Component。

Options

new Modal()
参数类型默认值描述
dataobject绑定属性 | Binding Properties
data.titlestring'提示'对话框标题 | Title of Dialog
data.contentstring''对话框内容
data.okButtonstring | booleantrue是否显示确定按钮。值为`string`时显示该段文字。
data.cancelButtonstring | booleanfalse是否显示取消按钮。值为`string`时显示该段文字。
data.widthnumbernull对话框宽度。值为否定时宽度为CSS设置的宽度。
data.classstring''补充class

Methods

modal.close(result)

关闭模态对话框

参数类型默认值描述
resultboolean点击确定还是取消

Static Methods

Modal.alert(content[, title])

弹出一个alert对话框。关闭时始终触发确定事件。

参数类型默认值描述
contentstring''对话框内容
titlestring'提示'对话框标题
Modal.confirm(content[, title])

弹出一个confirm对话框

参数类型默认值描述
contentstring''对话框内容
titlestring'提示'对话框标题

Events

modal.$on('close')

关闭对话框时触发

属性类型描述
resultboolean点击了确定还是取消
modal.$on('ok')

确定对话框时触发

属性类型描述
无参数
modal.$on('cancel')

取消对话框时触发

属性类型描述
无参数
