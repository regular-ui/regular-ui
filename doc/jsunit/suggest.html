自动提示 Suggest

<suggest source={source} />

    template: template,
    data: {
        source: [

严格模式

当为严格模式时，value属性必须在source中选择，否则为空。

<p><suggest source={source} placeholder="非严格模式" value={value1} /> {value1}</p>
<p><suggest source={source} placeholder="严格模式" strict={true} value={value2} /> {value2}</p>

远程数据

支持远程过滤。

<suggest service={@(this.service)} />

var component = new RGUI.Component({
    template: template,
    service: {
        getList: function(params, success) {
            RGUI.ajax.request({
                url: '../data/suggest.json',
                method: 'get',
                type: 'json',
                data: params,
                success: success
            });
        }
    }
});

API

Class

Suggest

继承自Dropdown。

Options

new Suggest()
参数类型默认值描述
dataobject绑定属性
data.sourceobject[][]数据源
data.source[].idnumber每项的id
data.source[].namestring每项的内容
data.selectedobjectnull当前选择项
data.valuestring''文本框中的值
data.placeholderstring'请输入'文本框的占位文字
data.minLengthnumber0最小提示长度。当输入长度>=该值后开始提示
data.matchTypestring'all'匹配方式，`all`表示匹配全局，`start`表示只匹配开头，`end`表示只匹配结尾
data.strictbooleanfalse是否为严格模式。当为严格模式时，`value`属性必须在source中选择，否则为空。
data.readonlybooleanfalse是否只读
data.disabledbooleanfalse是否禁用
data.visiblebooleantrue是否显示
data.classstring''补充class
serviceobject数据服务

Methods

suggest.select(item)

选择某一项

参数类型默认值描述
itemobject选择项
suggest.toggle(open)

在展开状态和收起状态之间切换

参数类型默认值描述
openboolean展开还是收起

Inherited Methods

@override suggest.select(item)
@override suggest.toggle(open)
suggest.$updateSource()

从service中更新数据源

参数类型默认值描述
无参数

属性类型描述

suggest.$on('toggle')

展开或收起状态改变时触发

属性类型描述
openboolean展开还是收起
