数字输入 NumberInput

示例

基本形式

<numberInput />
<numberInput value=6 />

禁用组件

<numberInput disabled={true} />

最大值和最小值

<numberInput min=5 max=8 />

数据绑定

<numberInput value={number} />
<numberInput value={number} min=5 max=12 />

事件

请打开浏览器的控制台查看结果。

<numberInput on-change={console.log('on-change:', '$event.value:', $event.value)} />

API

Class

NumberInput

继承自Input2。

Options

new NumberInput()
参数类型默认值描述
dataobject绑定属性
data.valuestring0输入框的值
data.typestring''输入框的类型
data.minnumberundefined最小值
data.maxnumberundefined最大值
data.readonlybooleanfalse是否只读
data.disabledbooleanfalse是否禁用
data.visiblebooleantrue是否显示
data.classstring''补充class

Methods

numberInput.add(value)

调整数值

参数类型默认值描述
valuenumber0加/减的值
numberInput.isOutOfRange(value)

是否超出规定的数值范围

参数类型默认值描述
valuenumber待测的值

Events

numberInput.$on('change')

数值改变时触发

属性类型描述
valuenumber改变后的数值
