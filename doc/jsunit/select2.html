选择扩展 Select2

示例

基本形式

<select2 source={source} />

var component = new RGUI.Component({
    template: template,
    data: {
        source: [
            {name: '选项1'},
            {name: '选项2'},
            {name: '选项3'}
        ]
    }
});

禁用某一项，禁用组件

<select2 source={source} />
<select2 source={source} disabled={true} />

var component = new RGUI.Component({
    template: template,
    data: {
        source: [
            {name: '选项1'},
            {name: '选项2'},
            {name: '选项3（禁用）', disabled: true}
        ]
    }
});

设置或取消默认项

<select2 source={source} placeholder="全部" />
<select2 source={source} placeholder="" />

var component = new RGUI.Component({
    template: template,
    data: {
        source: [
            {name: '选项1'},
            {name: '选项2'},
            {name: '选项3'}
        ]
    }
});

远程数据

<select2 service={@(this.service)} />

var component = new RGUI.Component({
    template: template,
    service: {
        getList: function(params, success) {
            RGUI.ajax.request({
                url: '../data/list.json',
                method: 'get',
                type: 'json',
                data: params,
                success: success
            });
        }
    }
});

数据绑定

<select2 source={source} selected={selected} /> 当前的选择项：{selected ? selected.name : 'null'}

var component = new RGUI.Component({
    template: template,
    data: {
        source: [
            {name: '选项1'},
            {name: '选项2'},
            {name: '选项3'}
        ]
    }
});

API

Class

Select2

继承自Dropdown。

Options

new Select2()
参数类型默认值描述
dataobject绑定属性
data.sourceobject[][]数据源
data.source[].namestring每项的内容
data.selectedobjectnull当前选择项
data.placeholderstring'请选择'默认项的文字
data.readonlybooleanfalse是否只读
data.disabledbooleanfalse是否禁用
data.visiblebooleantrue是否显示
data.classstring''补充class
serviceobject数据服务

Methods

select2.select(item)

选择某一项

参数类型默认值描述
itemobject选择项

Inherited Methods

select2.toggle(open)

在展开/收起状态之间切换

参数类型默认值描述
openboolean展开/收起
@override select2.select(item)
select2.$updateSource()

从service中更新数据源

参数类型默认值描述
无参数

Events

select2.$on('change')

选择项改变时触发

属性类型描述
selectedobject改变后的选择项
select2.$on('select')

选择某一项时触发

属性类型描述
selectedobject当前选择项

Inherited Events

select2.$on('toggle')

展开/收起时触发

属性类型描述
openobject展开/收起状态
@override select2.$on('select')
