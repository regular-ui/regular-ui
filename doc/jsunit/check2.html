多选按钮 Check2

示例

基本形式

<check2 name="多选按钮" />

半选状态

<check2 name="半选状态" checked={test} />

var component = new RGUI.Component({
    template: template,
    data: {
        test: null
    }
});

禁用组件

<check2 name="多选按钮" disabled />

API

Class

Check2

继承自Component。

Options

new Check2()
参数类型默认值数据流向描述
dataobject绑定属性
data.namestring''outer => inner多选按钮的文字
data.checkedbooleanfalseouter <=> inner多选按钮的选择状态。`false`表示未选，`true`表示已选，`null`表示半选。
data.blockbooleanfalseouter => inner是否以block方式显示
data.readonlybooleanfalseouter => inner是否只读
data.disabledbooleanfalseouter => inner是否禁用
data.visiblebooleantrueouter => inner是否显示
data.classstring''outer => inner补充class

Methods

check2.check(checked)

改变选中状态

参数类型默认值描述
checkedboolean选中状态。则在true/false之间切换。
返回值类型描述
无返回值

Events

check2.$on('change')

选中状态改变时触发

属性类型描述
sourceobject事件发起对象
dateobject改变后的选中状态
check2.$on('check')

改变选中状态时触发

属性类型描述
sourceobject事件发起对象
checkedboolean选中状态
